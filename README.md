# react-native-gesture-image-view

Image view with gestures

## Installation

```sh
npm install react-native-gesture-image-view
```

## Usage

```js
import { GestureImageViewView } from "react-native-gesture-image-view";

// ...

<GestureImageViewView color="tomato" />
```
